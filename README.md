# Storing-Diabetes-data-set-for-future-Ml
Steps to store and organize a diabetes dataset in Azure Machine Learning, using Blob storage and the Python SDK for easy access and future ML tasks.
In this lab, we organized and stored a diabetes dataset using Azure Machine Learning. First, we created a dedicated Storage Account connected to our workspace. Inside this account, we set up a Blob storage container to hold our data assets. We also created a File share for any code or notebooks. Next, we used the Python SDK to connect to the storage and register the dataset as a data asset. By normalizing and aggregating the data, we ensured it was structured consistently, making it ready for future machine learning tasks. This process ensures scalable, secure, and organized access to our data assets
